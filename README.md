Music Gallery
=============

## About
Converting an old jQuery project to AngularJS.

## Technology Used
* AngularJS
* Jade
* Stylus
* Grunt
* Bower

## Installation

Install package.json
	
	sudo npm install

Update dependencies
	
	bower update

Grunt Development
	
	grunt

Grunt Deploy
	
	grunt deploy
